Quiz question 6 : What is the smallest \(n\) such that a random variable drawn from a Geometric distribution with parameter prob=0.2 is less or equal to \(n\) with probability greater or equal to \(0.95\)?

Answer : The smallest \(n\) is 13.

ns_cutoff <- 0:100 #100 arbitrary, large number
ns_cutoff[min( which(pgeom(ns_cutoff, prob=0.2) >= 0.95))]
## [1] 13

An alternative way to arrive at the same result is using the qgeom function:

qgeom(p = 0.95, prob = 0.2)
## [1] 13

Quiz question 9 : Find by simulations using the var function what the variance of the Poisson distribution with parameter lambda = 5 is ( +/- 0.3 deviation accepted). For example you might want to generate 1000 instances of a Poisson(5) random variable.

Answer : The variance is about 5.

num_trials <- 50000
var(rpois(n = num_trials, lambda = 5))
## [1] 5.00235

Note that the theoretical variance of Poisson variables is equal to lambda parameter, so we got a consistent answer.

Quiz question 10 : Now instead consider the sum of 50 independent Poisson (0.1) random variables. Again using simulation (or otherwise), find the variance of this (again within +/- 0.3 range accepted).

Answer : The variance is about 5.

num_trials <- 50000
var(replicate(num_trials, sum(rpois(n = 50, lambda = 0.1))))
## [1] 4.978917

Note, that we again get the same answer. This is because, a sum of Poisson variables, \(X_i \sim Pois(\lambda_i), \quad i = 1, \dots, M\) is Poisson distributed \(Y = \sum_{i = 1}^M X_i \sim Pois(\sum_{i = 1}^M\lambda_i)\)

Quiz Question 14 : Based on these results, do you believe that the nucleotide frequencies in the C.elegans chrM sequence are all the same?

# nucleotide frequency of chrM
lfM = letterFrequency(Celegans$chrM, letters=c("A", "C", "G", "T"))
lfM
##    A    C    G    T 
## 4335 1225 2055 6179

We’re going to compute a statistic that measures how close two multionial outputs are to each other as sum of squared deviation from expected frequencies over expected frequencies. We’ll take the average squared difference between expected (e) and observed (o) counts, scaled by e. We will call the function oestat:

oestat = function(o, e){
  sum( (e-o)^2/e )
}
n = length(Celegans$chrM)
expected = rep(n/4 ,4)
oe = oestat( o = lfM, e=expected)
oe
## [1] 4386.634

Then, we compute the distribution of the same statistics assuming it came from the null distribution if all nucleotides were equally likely:

set.seed(1)
B = 10000
oenull = replicate(B, oestat(e=expected, o=rmultinom(1,size = n, prob = rep(1/4,4))))
hist(oenull, breaks = 50)

quantile(oenull, p = 0.95)
##      95% 
## 7.754444
oe
## [1] 4386.634
oe > quantile(oenull, p = 0.95)
##  95% 
## TRUE

Answer : The observed statistic ‘oe’ is much bigger than the 95% quantile of the empirical null distribution of the statistics, so we can conclude that Celegans’s chrM does not have an even distribution of the nucleotides.
